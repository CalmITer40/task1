## Развертывание

- клонировать репозиторий/запустить контейнер
- установить зависимости
- создать конфиг на основе **.env.example**
- выполнить миграции **artisan migrate**
- запустить очередь **artisan queue:work**
- собрать фронт **npm run dev**
